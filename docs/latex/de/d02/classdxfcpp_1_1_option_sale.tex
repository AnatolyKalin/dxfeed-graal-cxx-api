\doxysubsubsection{Detailed Description}
Option Sale event represents a trade or another market event with the price (for example, market open/close price, etc.) for each option symbol listed under the specified \doxylink{classdxfcpp_1_1_underlying}{Underlying}. Option Sales are intended to provide information about option trades {\bfseries{in a continuous time slice}} with the additional metrics, like Option Volatility, Option Delta, and \doxylink{classdxfcpp_1_1_underlying}{Underlying} Price.

Option Sale events have unique \doxylink{classdxfcpp_1_1_option_sale_ae8097922d7f01518786795c3709c9db6}{index} which can be used for later correction/cancellation processing.

Option sale data source provides a consistent view of the set of known option sales. The corresponding information is carried in \doxylink{classdxfcpp_1_1_option_sale_acade17b388636061a96a69aeab5cb785}{event\+Flags} property. The logic behind this property is detailed in \doxylink{structdxfcpp_1_1_indexed_event}{Indexed\+Event} class documentation. Multiple event sources for the same symbol are not supported for option sale events, thus \doxylink{classdxfcpp_1_1_option_sale_af2ab852e87718d4e703ddf169eb43d0c}{source} property is always \doxylink{classdxfcpp_1_1_indexed_event_source_ab7bc1a5037c8e18b2adeb45237cea5eb}{DEFAULT}.

This event is implemented on top of QDS records {\ttfamily \doxylink{classdxfcpp_1_1_option_sale}{Option\+Sale}}.
