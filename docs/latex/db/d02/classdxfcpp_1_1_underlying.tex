\doxysubsubsection{Detailed Description}
\doxylink{classdxfcpp_1_1_underlying}{Underlying} event is a snapshot of computed values that are available for an option underlying symbol based on the option prices on the market. It represents the most recent information that is available about the corresponding values on the market at any given moment of time.

\doxyparagraph*{\label{classdxfcpp_1_1_underlying_eventFlagsSection}%
\Hypertarget{classdxfcpp_1_1_underlying_eventFlagsSection}%
Event flags, transactions and snapshots}

Some \doxylink{classdxfcpp_1_1_underlying}{Underlying} sources provide a consistent view of the set of known \doxylink{classdxfcpp_1_1_underlying}{Underlying} events. The corresponding information is carried in \doxylink{classdxfcpp_1_1_underlying_a0712a866c779a2dcb05c70cb940486b4}{event\+Flags} property. The logic behind this property is detailed in \doxylink{structdxfcpp_1_1_indexed_event}{Indexed\+Event} class documentation. Multiple event sources for the same symbol are not supported for \doxylink{classdxfcpp_1_1_underlying}{Underlying}, thus \doxylink{classdxfcpp_1_1_underlying_a39751b0252b25e18f437a7cd29c62fa4}{source} property is always \doxylink{classdxfcpp_1_1_indexed_event_source_ab7bc1a5037c8e18b2adeb45237cea5eb}{DEFAULT}.

Time\+Series\+Event\+Model class handles all the snapshot and transaction logic and conveniently represents a list current of time-\/series events order by their \doxylink{classdxfcpp_1_1_underlying_a55e87e436c21ec82cf9990b6ca97e98b}{time}.

\doxyparagraph*{Publishing \doxylink{classdxfcpp_1_1_underlying}{Underlying}}

Publishing of \doxylink{classdxfcpp_1_1_underlying}{Underlying} events follows the general rules explained in \doxylink{structdxfcpp_1_1_time_series_event}{Time\+Series\+Event} class documentation.

\doxyparagraph*{Implementation details}

This event is implemented on top of QDS record {\ttfamily \doxylink{classdxfcpp_1_1_underlying}{Underlying}}.
