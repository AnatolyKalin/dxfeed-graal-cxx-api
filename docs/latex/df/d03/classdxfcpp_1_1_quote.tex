\doxysubsubsection{Detailed Description}
\doxylink{classdxfcpp_1_1_quote}{Quote} event is a snapshot of the best bid and ask prices, and other fields that change with each quote. It represents the most recent information that is available about the best quote on the market at any given moment of time.

Bid corresponds to the best (maximal price) order to buy, ask corresponds to the best (minimal price) order to sell.
