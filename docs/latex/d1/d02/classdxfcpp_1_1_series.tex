\doxysubsubsection{Detailed Description}
\doxylink{classdxfcpp_1_1_series}{Series} event is a snapshot of computed values that are available for all option series for a given underlying symbol based on the option prices on the market. It represents the most recent information that is available about the corresponding values on the market at any given moment of time.

\doxylink{classdxfcpp_1_1_series}{Series} is an \doxylink{structdxfcpp_1_1_indexed_event}{Indexed\+Event} with multiple instances of event available for each underlying symbol. Each series event instance corresponds to an Option\+Series of the corresponding underlying. The correspondence between a series event instance and an Option\+Series is established via \doxylink{classdxfcpp_1_1_series_a67dc5b1995c961013c60fe129c57f7d4}{expiration} property. If case where there are multiple series at the same expiration day id, then series events are are ordered by their \doxylink{classdxfcpp_1_1_series_a8f61e55dbd45c37621f74f9d066a14ac}{index} in the same order as the corresponding Option\+Series are ordered by their attributes.

\doxyparagraph*{\label{classdxfcpp_1_1_underlying_eventFlagsSection}%
\Hypertarget{classdxfcpp_1_1_underlying_eventFlagsSection}%
Event flags, transactions and snapshots}

\doxylink{classdxfcpp_1_1_series}{Series} data source provides a consistent view of the set of known series. The corresponding information is carried in \doxylink{classdxfcpp_1_1_series_aa094b0a6f3258a023d9e33693a08005e}{event\+Flags} property. The logic behind this property is detailed in \doxylink{structdxfcpp_1_1_indexed_event}{Indexed\+Event} class documentation. Multiple event sources for the same symbol are not supported for series, thus \doxylink{classdxfcpp_1_1_series_ad4864569fa4661be1f26c0e30073a55c}{source} property is always \doxylink{classdxfcpp_1_1_indexed_event_source_ab7bc1a5037c8e18b2adeb45237cea5eb}{DEFAULT}.

Indexed\+Event\+Model class handles all the snapshot and transaction logic and conveniently represents a list current of events.

\doxyparagraph*{Implementation details}

This event is implemented on top of QDS records {\ttfamily \doxylink{classdxfcpp_1_1_series}{Series}}.
